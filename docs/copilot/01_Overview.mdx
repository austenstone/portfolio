# Copilot Customization Cheat Sheet

GitHub Copilot isn't just autocomplete anymore. It's a customizable AI coding system with **7 distinct extension points** — from always-on instructions to full external tool integrations.

This guide covers all of them: what they do, where the files go, which surfaces support them, and when to reach for each one.

## Quick Decision Guide

- **Want consistent coding style across the team?** → [Custom Instructions](./Custom%20Instructions)
- **Have a repeatable task (scaffolding, tests)?** → [Prompt Files](./Prompt%20Files)
- **Need a specialist reviewer or advisor?** → [Custom Agents](./Custom%20Agents)
- **Multi-step workflows with scripts?** → [Agent Skills](./Agent%20Skills)
- **Auto-format or lint after every edit?** → [Hooks](./Hooks)
- **Connect to external APIs or databases?** → [MCP Servers](./MCP%20Servers)
