# React Chat

A **real-time chat application** built with React and Firebase, demonstrating modern web development practices and real-time communication technologies.

## Overview

React Chat is a simple yet powerful chatroom application that showcases the integration of React with Firebase for real-time messaging. It provides a clean, responsive interface for group conversations with features you'd expect from a modern chat application.

## Live Application

**Try It Now:** [chat.austen.info](https://chat.austen.info)

Join the conversation and experience real-time messaging in action!

## Key Features

### Real-time Messaging
- **Instant Delivery** - Messages appear immediately across all connected clients
- **Live Typing Indicators** - See when others are typing
- **Online Presence** - Know who's currently in the chat
- **Message History** - Persistent chat history across sessions

### User Experience
- **Clean Interface** - Minimalist design focused on conversation
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Emoji Support** - Express yourself with emoji reactions

### Modern Features

### Development Setup
1. **Clone Repository** - Download the source code
2. **Install Dependencies** - Run `npm install` to set up packages
3. **Firebase Configuration** - Set up Firebase project and credentials
4. **Environment Variables** - Configure API keys and settings
5. **Development Server** - Run `npm start` for local development
6. **Build & Deploy** - Use `npm run build` for production deployment
